Introduction

Welcome to Handling Dynamic Assets - example 1

In this folder you will find a zip of the Experience - you can import this into Vuforia Studio
Two barcode images are included - you will use these to scan the ID of the models

Two models are provided, QuadcopterDA1 (basic) and QuadcopterDA2 (advanced)
A Creo Illustrate _start project is also included if you want to learn how to create the basic model.


A folder including a 3d printable quadcopter is included.
